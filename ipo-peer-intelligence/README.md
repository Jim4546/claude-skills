# ipo-peer-intelligence

IPO 同行与竞对尽调 skill。它从第三方客观视角，对拟上市公司、已递表公司、同行、竞对、供应商、客户或潜在投资标的做证据优先的产业研究和买方尽调分析。

## 适合什么场景

- 看一家拟上市公司或辅导备案公司，需要快速判断基本面和风险。
- 对某个同行、竞对、供应商或客户做桌面尽调。
- 阅读招股书、问询函、年报和公开资料后，需要整理成中文分析报告。
- 希望得到分级红旗清单、证据缺口和后续访谈问题。

## 工作原则

- 准确性高于完整性，没有证据的数字不编。
- 每个关键结论区分事实、推断、风险和问题。
- 公开资料没有披露的内容必须标注“待核实”。
- 默认中文输出，英文材料保留关键术语并解释。
- 视角是客观研究，不做投资推介、包装或竞对攻击。

## 两种深度

| 档位 | 适用场景 | 输出 |
|---|---|---|
| 快速筛查 | 初筛、大量标的快速过一遍 | 公司画像、关键假设、分级红旗、待核实问题 |
| 完整尽调 | 重点标的、对外汇报、投前研究 | 十大模块展开、表格、证据台账、红旗清单、后续访谈问题和可视化 |

如果用户没有指定深度，skill 会先确认要做快速筛查还是完整尽调。

## 常用说法

- “帮我做一下这家公司的 IPO 尽调：<公司名>”
- “看一下这家公司招股书里有哪些红旗”
- “帮我做同行对标和竞对分析：<公司 A> vs <公司 B>”
- “基于这些材料做一份投前桌面尽调”

## 注意事项

- 只使用可验证资料，不把传闻当事实。
- 财务、市场份额、产能、客户收入等数字必须能回到来源。
- 没有访谈材料时，不假装做过客户、供应商或专家访谈。
- 对监管问询、关联交易、收入真实性、毛利率异常等问题要单独标注。
