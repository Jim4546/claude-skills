# 安装指南_图文

这份指南面向第一次安装 `ai-workflow-skills` 的同事或新机器。照着做一遍，以后更新只需要 `git pull` 再运行部署脚本。

## 1. 准备工具

需要先确认本机已经安装：

- Git
- Claude Code
- PowerShell 或 Git Bash

如果要使用飞书相关 skill，还需要安装并登录 `lark-cli`。如果要使用网页抓取、公众号归档、SPA 导出等 skill，还需要 Python 3.10+。

## 2. 克隆仓库

打开 PowerShell 或 Git Bash，选择一个你想放技能库的位置，例如桌面：

```bash
git clone https://github.com/Jim4546/ai-workflow-skills.git
cd ai-workflow-skills
```

## 3. 启用自动部署钩子

```bash
git config core.hooksPath hooks
```

启用后，每次提交或拉取更新，仓库会自动把所有 skill 复制到 `~/.claude/skills/`。

## 4. 首次部署

Git Bash、macOS 或 Linux：

```bash
bash deploy.sh
```

Windows PowerShell：

```powershell
powershell -ExecutionPolicy Bypass -File .\deploy.ps1
```

看到类似下面的输出就说明部署成功：

```text
deployed devflow -> C:\Users\<用户名>\.claude\skills\devflow
deployed feishu-doc-mirror -> C:\Users\<用户名>\.claude\skills\feishu-doc-mirror
```

## 5. 重启或刷新 Claude Code

部署完成后，重启 Claude Code，或者让当前会话刷新 skills。之后直接用中文描述需求即可，例如：

```text
帮我把这篇公众号转到知识库：<URL>
```

## 6. 安装可选依赖

不同 skill 的依赖不同，第一次用到时按提示安装即可。常见依赖如下：

### Playwright

用于浏览器渲染、反爬抓取、飞书图片恢复。

```bash
pip install playwright
playwright install chromium
```

### Python 依赖

某些 skill 有自己的 `requirements.txt`：

```bash
pip install -r feishu-doc-mirror/requirements.txt
pip install -r spa-to-pdf/requirements.txt
pip install -r wechat2lark/scripts/requirements.txt
```

### lark-cli 登录

飞书相关 skill 需要先登录：

```bash
lark-cli auth login --domain docs
```

需要访问飞书即时消息、日历、妙记时，根据 skill 提示补充相应授权。

## 7. 日常更新

进入仓库目录：

```bash
git pull
bash deploy.sh
```

PowerShell：

```powershell
git pull
powershell -ExecutionPolicy Bypass -File .\deploy.ps1
```

## 8. 常见问题

### Claude Code 没识别到 skill

先确认 `~/.claude/skills/<skill-name>/SKILL.md` 是否存在。不存在就重新运行部署脚本。

### 运行脚本提示缺 Python 包

按该 skill 目录里的 `requirements.txt` 安装依赖。

### 飞书操作失败

先运行：

```bash
lark-cli auth status
```

如果没有登录或缺少权限，重新 `auth login`。

### Windows 运行 PowerShell 脚本被拦截

使用带执行策略的命令：

```powershell
powershell -ExecutionPolicy Bypass -File .\deploy.ps1
```

## 9. 给维护者

- 只改本仓库，不直接改 `~/.claude/skills/` 下的副本。
- 新增 skill 时保持“根目录 + SKILL.md”的结构。
- README 用中文写，面向使用者说明能做什么、怎么触发、依赖和限制。
- 改完提交后，hooks 会自动部署到本机。
