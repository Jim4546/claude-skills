# `+总结本次` 工作流

**目的**：在某个 Claude 对话工作告一段落时调用，把本次会话写成一段日报片段，存到本地，供日终 `+发布日报` 汇总。

## 步骤

### 1. 实质交付自评（重要！空对话直接拒绝）

回顾本次对话，判断是否产生了以下任一类有效产出：
- 代码改动（写过 / 改过文件，无论是否 commit）
- 文档撰写或修改
- 明确的决策、方案、设计选型
- 数据处理 / 配置变更
- 排查得出的结论（即使没动代码）

若**全部为否**（纯查文档 / 纯闲聊 / 中途打断没做完什么） → 直接告诉用户：

> 本次对话无实质交付，未记录到日报。

**不要**写片段文件。流程结束。

### 2. 识别项目

获取 cwd（PowerShell：`$PWD.Path`），从 cwd 开始**向上递归查找**：
- 优先匹配 `.git` 目录
- 其次匹配 `CLAUDE.md` 文件
- 找到任一即停，该目录的**目录名**就是 `project`
- 一直到根都找不到 → `project = "__misc"`

> 注意 Windows 路径分隔符；使用 PowerShell 时 `Test-Path "$dir\.git"` 和 `Test-Path "$dir\CLAUDE.md"`。

### 3. 生成片段内容

按"完成事项 / 收获与判断 / 下一步"三要素总结，每个独立子任务一个 `### N. 标题` 段落：

```markdown
### 1. <一句话标题，描述这件工作本身，不带"我"或"Claude">
- 完成事项：<做了什么 + 量化产出；多条用嵌套 `  - ` 列出。涉及代码附 commit hash + 已 push / 未提交>
- 收获与判断：<关键决策、技术选型、踩到的坑、形成的认知>
- 下一步：<后续计划 / 待办；无则填"（无）">
```

#### 口吻硬约束（重要 — 这是对外汇报，不是对话回顾）

- **禁止第一人称**：不写"我"、"我让"、"我决定"
- **禁止把 Claude 出场**：不写"Claude 帮我"、"由 AI 协助"、"和 AI 一起"
- **禁止把用户当对话方**：不写"用户提到"、"用户表态"、"经讨论"
- **主语是项目 / 模块 / 工作本身**：用"完成 X"、"新增 Y"、"事故复盘表明 Z"
- **避免对话化口语**：不写"OK"、"那么"、"其实"、"是不是"等

**对照示例**：
- ❌ "我已 git add 完 v3.0e 9 个文件等待 commit 时，v3.0d 窗口抢先 commit"
- ✅ "并行 commit 期间检测到 schema.prisma 被另一分支 commit 覆盖，导致本批 status/extractError 字段丢失"
- ❌ "用户主动表态'代码 OK 就行'后才写 marker"
- ✅ "verify 报告标注'部分通过 + 已知限制本机无法跑浏览器真机验收'，由业务方接收 UI 验收责任后写 marker 放行"

#### 内容要求

- 标题 N 从 1 开始连续编号
- 只写**结果 + 决策**，不写过程八卦
- 涉及代码：commit hash + push 状态必写
- 多个独立工作分多个段落，不要塞到一个里
- 量化优先（行数、文件数、commit 数、字段数等）

### 4. 写文件

路径：`C:\Users\jiawei.wang\.claude\data\daily-report\<YYYY-MM-DD>\<project>__<HH-mm>.md`

- 日期、时间用**本地时间**（北京时间）
- 父目录不存在用 `New-Item -ItemType Directory -Force -Path ...` 创建
- frontmatter + 正文：

```markdown
---
project: <project_name>
working_dir: <绝对路径>
captured_at: <ISO 8601 带时区，如 2026-05-19T14:30:00+08:00>
---

### 1. ...
...
```

- 同一窗口反复跑 → 每次新文件（时间戳不同），不要去 append 旧文件，避免冲突

### 5. 输出

```
已记录到 C:\Users\jiawei.wang\.claude\data\daily-report\2026-05-19\cvabase__14-30.md
今天该项目已累计 N 个片段。
```

> N 通过 `Get-ChildItem` 数当日同项目片段数得到。

## 项目名特殊情况

- `__misc` 项目：发布时会合并为「杂项」一节
- 项目名含空格 / 中文：原样保留，文件名安全（Windows 文件名禁止字符替换为 `_`）
- 同一目录两个对话窗口同时跑 → 时间戳精确到分钟，若撞了，追加 `-2` 后缀
