# 使用手册_话术

这份手册面向日常使用者：不用记 skill 名，直接用中文自然语言说明“想做什么”。Claude Code 会根据 `SKILL.md` 自动选择合适的 skill。

## 开发流程

### devflow

- 能干什么：把一个开发需求按 idea、plan、执行、验证、简化和提交的节奏推进，减少需求、文档和代码漂移。
- 怎么说：“这个功能走 devflow，先帮我梳理 plan。”
- 会得到什么：需求拆解、实施计划、验收点和后续执行步骤。
- 真实例子：“我要做一个导入 Excel 的功能，帮我走一遍 devflow。”
- 小提示：适合中等以上改动；小修小补可以直接说明要改什么。

## 日报与复盘

### daily-report

- 能干什么：汇总当天多个 Claude Code 窗口的工作，发布到飞书开发日报。
- 怎么说：“写今天的日报并发布到飞书。”
- 会得到什么：按项目分组的日报文档。
- 真实例子：“把今天所有 Claude Code 工作整理成开发日报。”
- 小提示：更偏完整记录，不等于对外精简汇报。

### personal-worklog

- 能干什么：写个人工作小结，强调你如何拆需求、拍板、纠正 AI 和把控节奏。
- 怎么说：“写今天的个人 worklog。”
- 会得到什么：偏个人贡献视角的工作总结。
- 真实例子：“总结我今天指挥 AI 做了什么。”
- 小提示：适合自我复盘，不适合作为对外日报直接发送。

### work-review

- 能干什么：一次扫本地对话、飞书会话和今日会议，生成对外汇报用的精简日报。
- 怎么说：“写今天的精简日报。”
- 会得到什么：`王佳玮工作日报_YYYYMMDD` 飞书文档和群消息。
- 真实例子：“jjrb，按今天所有项目进展出一版精简日报。”
- 小提示：它默认直接发布；需要调试时加 `--dry-run`。

## 研究尽调

### entity-dossier

- 能干什么：做企业、人物、投资方或关联实体的信息整理稿。
- 怎么说：“帮我做一下这家公司和创始人的背调。”
- 会得到什么：含来源、股权、任职时间线和图示的飞书文档。
- 真实例子：“$entity-dossier，查一下某某公司、创始人和主要投资方。”
- 小提示：适合公开资料背调；高敏结论要保留来源和不确定性。

### ipo-peer-intelligence

- 能干什么：做拟上市公司、同行、竞对、供应商或客户的桌面尽调。
- 怎么说：“帮我做这家公司的 IPO 尽调。”
- 会得到什么：公司画像、红旗清单、证据缺口、同行对标和后续问题。
- 真实例子：“基于招股书和问询函，分析这家公司有哪些风险。”
- 小提示：没有证据的数字不要催它补全，应标“待核实”。

## 内容归档

### article-origin-finder

- 能干什么：给文章链接找原文、查证来源，尤其适合中文编译稿找英文原文。
- 怎么说：“帮我找这篇文章的英文原文并核对。”
- 会得到什么：原文信息表、英文原文全文和中英核对表。
- 真实例子：“这篇公众号文章是不是翻译的？帮我溯源。”
- 小提示：如果没有英文原文，它会明确写“未找到”，不会硬编。

### feishu-doc-mirror

- 能干什么：把飞书分享文档或社区文章完整镜像到自己的空间。
- 怎么说：“这个飞书文档不能复制，帮我克隆一份。”
- 会得到什么：新的可编辑飞书 docx 链接。
- 真实例子：“把这篇飞书社区文章镜像到我的知识库。”
- 小提示：图片恢复要求源文档能匿名访问。

### wechat2lark

- 能干什么：把微信公众号文章转成飞书云文档并归档到知识库。
- 怎么说：“公众号转飞书：<URL>。”
- 会得到什么：新的飞书 wiki 链接。
- 真实例子：“帮我把这篇公众号文章归档到团队知识库。”
- 小提示：关注后可见的私密文章不支持。

## 文档转换

### spa-to-pdf

- 能干什么：把 JS 渲染的 SPA 网页导出成 Word 和 PDF。
- 怎么说：“把这个网页所有章节整理成 Word 和 PDF。”
- 会得到什么：`output.docx` 和 `output.pdf`。
- 真实例子：“这个 `#/page` 页面 curl 抓不到，帮我浏览器渲染后导出。”
- 小提示：PDF 步骤依赖 Windows + Microsoft Word。

## 会议处理

### minutes-clip-resummary

- 能干什么：裁剪长录音或飞书妙记片段，重新生成智能会议纪要。
- 怎么说：“把这段妙记裁掉无关部分后重做纪要。”
- 会得到什么：新的妙记链接和重新生成的纪要。
- 真实例子：“保留 12:30 到 45:00 这一段，重新生成智能纪要。”
- 小提示：需要明确保留或删除的时间段。

## 图片处理

### image-recolor-transparent

- 能干什么：按模板色系改图片配色，并抠成透明底。
- 怎么说：“按这个 PPT 模板色系给图片改色，白底抠透明。”
- 会得到什么：透明底图片和双底色预览。
- 真实例子：“把图里的红色改成橙色，灰黑改成蓝灰。”
- 小提示：复杂图片要看预览确认边缘是否干净。
